# Company Productivity Analysis

This is a Python project that analyzes employee productivity using NumPy and Matplotlib.

---

## Files

- main.py – runs the program  
- operationclass.py – contains calculations and plots  
- company.txt – input data  

---

## What This Project Does

- Reads data from a text file  
- Finds best and worst performing companies  
- Calculates average productivity  
- Computes salaries  
- Plots employee productivity
